# Fleming's Left Hand Rule Interactive App

An interactive drilling app for HKDSE students to learn and practice Fleming's Left Hand Rule with 3D visualization.

## Features

- 3D interactive hand model that can be rotated freely
- Random question generation covering all combinations
- Immediate feedback on answers
- Continuous drilling mode

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This app is optimized for deployment on Vercel:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Deploy automatically

## Fleming's Left Hand Rule

- **Thumb**: Direction of Force/Motion (on the conductor)
- **First Finger**: Direction of Magnetic Field (North to South)
- **Second Finger**: Direction of Current (Conventional current)
